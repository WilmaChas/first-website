This website however is a part of our latest course - DevSecOps.
We have created a simple web application and learned how to take it from code to deployment.
We used Docker to package the app with everything it needs to run, and Kubernetes to manage it inside our team namespace in a shared cluster (although now we use render as the course is coming to an end and our cluster has been decommissioned).
Our CI/CD pipeline helps automate the process by building the app, handling container images, and making deployments smoother and more consistent.
We also worked with cloud infrastructure so the application could run outside our own computers in a more realistic environment.
Through this project, we’ve been learning how development, security, and operations all come together in one workflow.
